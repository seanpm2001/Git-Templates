The capital doesn't differentiate the 2 files by much. They both still function the same.

A .UDF file is a Universal Disk Format (UDF) optical disk image file. It contains the raw contents of a CD image in a digital format. It is a common file format. Some notable uses include installing software, listening to music, installing an operating system in a virtual machine/host machine, and more. This file extension is not to be confused with the file system that runs on it, known as UDF.

Definition 2: Pseudo-extension (null)
